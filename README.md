# Movie Search App 🎬

A movie search application that connects to the OMDb API 
to find and display real movie data.

## Features![Screenshot_3-4-2026_12345_terry-stang92 github io](https://github.com/user-attachments/assets/422db0c1-f5f8-47da-a0ee-dec2ef573b14)

- Live movie search
- Displays movie posters and titles
- Connects to external API (OMDb)
- Space-themed design

## Built With
- HTML5
- CSS3
- JavaScript (Fetch API)
