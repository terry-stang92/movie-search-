# Movie Search App 🎬

A movie search application that connects to the OMDb API 
to find and display real movie data.

## Features
- Live movie search
- Displays movie posters and titles
- Connects to external API (OMDb)
- Space-themed design

## Built With
- HTML5
- CSS3
- JavaScript (Fetch API)
